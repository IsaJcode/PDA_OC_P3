This code allows running the game :"Aidez MacGyver à s'échapper!"

INSTALLATION:
1-Install python 3.4
2-Install a virtual environnement
3-Install all required with the file requirements.txt
4-Start the file joue.py to play

RULES:
MacGyver is in a labyrinth and has to escape alive.
The exit is controlled by a keeper.
MacGyver could reach the exit but to go out alive, he should first collect all objets presents in the labyrinth.

Have fun.
